why i built this

I'm a student. For a long time the best coding help was locked behind a paycheck I didn't have, and that never sat right with me. So AXIOM is my attempt at the opposite: a real agent, not a toy, that's actually good on the models anyone can use for free. If it helps one person who'd otherwise be priced out, it was worth every late night.
